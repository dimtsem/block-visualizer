About


This aplication provides information about bitcoin (BTC) transactions using the data available on Blockchain.info. All the calculations are performed interactively and there is no database dependence, except the classifier models, which have been trained locally.

It provides four types of service, explained in the order in which they appear on the home page.

1. It creates a visualization of the transactions in any given individual block, and can also retrieve the latest block.

2. It creates a visualization of the transaction graph for any given wallet on the BTC blockchain.

3. It compares any two wallets on the BTC blockchain and returns an estimate about whether or not these wallets belong to the same entity. This estimate is based on the similarity algorithm described in this paper.

4. It takes a wallet and classifies it as a type of transactor according to a (unsupervised) classification algorithm described in more detail here.
